## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- React + Vite
- CSS3 (Custom styling)
- Lucide React Icons

## Features

- Light theme with maroon (#66261A) and cream (#fff4ef) color scheme
- Terminal-style hero section with typing animation
- Responsive design
- Project showcase
- GitHub integration with live contribution heatmap
- Contact form

## Development

The app runs on `http://localhost:5173` by default.

## Build

Production build is created in the `dist` folder.
